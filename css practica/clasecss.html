Elementos en bloque

En un modelo de bloque, cualquier elemento es un contenedor rectangular,
independientemente de cómo está representado visualmente en la página. El elemento tiene un área de contenido y campos opcionales, sangría y marco (borde).

This is the content of an element, such as text, image, video, or nested elements. By default, the height of the content area is determined by the content, and the width is determined by the element type.
